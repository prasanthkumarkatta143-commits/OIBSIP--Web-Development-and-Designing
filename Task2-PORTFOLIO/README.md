
## 📌 Project Overview

The Personal Portfolio Website is a simple, responsive, and attractive web page that acts as an online portfolio. It allows users to introduce themselves, display their technical skills, highlight sample projects, provide a downloadable resume, and share contact details.

This project is ideal for:
- Internship submission
- Beginner web development practice
- Student portfolio showcase
- Personal branding

---

## ✨ Features

- **Responsive Navigation Bar**
  - Sticky header with navigation links
  - Mobile-friendly menu toggle

- **Hero Section**
  - Personal introduction
  - Profile image
  - "Hire Me" button

- **About Section**
  - Short description about the portfolio owner

- **Skills Section**
  - Skill cards with hover effects

- **Projects Section**
  - Project cards with titles and descriptions

- **Resume Section**
  - Resume download button

- **Contact Section**
  - Email, phone number, and location details

- **Footer**
  - Copyright information

- **JavaScript Functionality**
  - Mobile menu toggle
  - Smooth user interaction

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website
- **CSS3** – Styling and layout
- **JavaScript** – Interactivity and mobile navigation

## 📄 Output Description

The output of this project is a **responsive personal portfolio website** with a clean and modern design.

### Sections included in the website:

1. **Header / Navigation Bar**

   * Displays navigation links such as Home, About, Skills, Projects, Resume, and Contact.

2. **Hero Section**

   * Shows the user’s name, role, short introduction, profile image, and a call-to-action button.

3. **About Section**

   * Contains a brief self-introduction and career objective.

4. **Skills Section**

   * Displays technical skills in styled cards.

5. **Projects Section**

   * Showcases example projects in project cards.

6. **Resume Section**

   * Includes a button to download the resume.

7. **Contact Section**

   * Provides contact details like email, phone, and location.

8. **Footer**

   * Displays copyright information.

## ▶️ How to Run the Project

1. Download or copy the project files.
2. Create a folder named **portfolio**.
3. Save the files:

   * `index.html`
   * `style.css`
   * `script.js`
4. Open `index.html` in any web browser.
