

## 📌 Project Objective

The main objective of this project is to build a visually appealing **Landing Page** that includes:

- A **Header / Navigation Bar**
- A **Hero Section**
- A **Features Section**
- An **About Section**
- A **Call-to-Action Section**
- A **Contact Form**
- A **Footer**

This project helps beginners understand how to divide a webpage into sections and style each part effectively.

---

## 🛠️ Technologies Used

- **HTML5** – for webpage structure
- **CSS3** – for styling and layout
- **JavaScript** – for interactivity

---

## 📄 File Description

### `index.html`

Contains the complete structure of the landing page including:

* Header
* Hero section
* Features section
* About section
* CTA section
* Contact section
* Footer

### `style.css`

Contains all the styling used in the project such as:

* Layout and spacing
* Typography
* Buttons and hover effects
* Responsive design
* Dark mode styles

### `script.js`

Adds interactive functionality such as:

* Dark mode toggle
* Contact form submission message

## 🚀 How to Run the Project

Follow these steps to run the project on your system:

1. **Download or clone** the project files
2. Place all files inside one folder named `landing-page`
3. Make sure the folder contains:

   * `index.html`
   * `style.css`
   * `script.js`
4. Open the `index.html` file in your browser
